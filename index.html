github 的使用步骤

  • 1.点击 https://github.com/ 网址进入GitHub官网注册一个账号
  • 2.注册完成，点击Create a new repository创建一个仓库
  • 3.点击 https://desktop.github.com/ 网址下载GitHub Desktop
  • 4.下载安装完成后，打开GitHub Desktop, 然后选择菜单栏中的File 点击 clone repositories
  • 5.点击 clone repositories 后的表单中选择 you repositories下第二步所创建好的仓库，最后点击 clone
